About the Project

Mudirdokan is an industry-standard e-commerce platform that enables users to order food and grocery items conveniently. Our goal is to provide a seamless online shopping experience, ensuring quality products and excellent customer service.

Key Features

  • User registration and authentication
  • Product catalog with detailed descriptions
  • Shopping cart functionality
  • Order placement and tracking
  • Secure payment processing

API Endpoints

Endpoint Method Description
/api/products GET Retrieve a list of all products available for sale.
/api/products/:id GET Retrieve detailed information about a specific product.
/api/cart POST Add an item to the shopping cart.
/api/orders POST Place a new order.
/api/users/register POST Create a new user account.
/api/users/login POST Authenticate a user and start a session.
